# 外贸内参汇总首页需求说明

## 目标

将 neicanhtml 站点的 `index.html` 从「单期日报副本」改为**汇总导航页**，实现：

1. 首页展示当日最新内参全文
2. 提供按月份浏览的情绪日历，可查看历史每天的外贸行业情绪并跳转到对应日报
3. 每次生成新内参并导出时，自动更新 index 与日历数据

## 功能范围

### 1. 当日最新内参

- 在汇总页上半部分内嵌展示**当日**最新一期内参全文
- 实现方式：`<iframe src="neican-YYYY-MM-DD.html">`，日期为 manifest 中的最新日期
- 若无数据则显示「暂无内参数据」提示

### 2. 情绪日历

- 默认展示当月日历，支持**上一月 / 下一月**切换
- 历史范围：从 **2026 年 3 月 16 日**起（首期内参日期）
- 有数据的日期：
  - 显示当日情绪词（如：焦虑、平淡、喜悦）
  - 使用与日报一致的情绪配色（焦虑红、愤怒紫、喜悦金等）做视觉区分
  - 点击跳转到 `neican-YYYY-MM-DD.html` 查看该日完整日报
- 无数据的日期：浅灰显示，不可点击
- 日历数据来源：前端通过 `fetch('./manifest.json')` 获取，纯静态无后端

### 3. 数据与更新

- **manifest.json**：由 waimaoneican 的 `deploy.py` 在每次导出时重建
  - 扫描 neicanhtml 目录下所有 `neican-YYYY-MM-DD.json`
  - 提取每期的 `date`、`overall_mood.word`、`overall_mood.reason` 及对应情绪颜色
  - 按日期倒序排列，写入 `manifest.json`
- **index.html**：由 `deploy.py` 的 `_render_index_html()` 在每次导出时重新生成
  - 结构：站头 + 当日内参 iframe + 情绪日历（JS 渲染）
- 每次执行 `run_daily.py` 并启用 Git 导出时，会复制本期 HTML/JSON 到 neicanhtml、重建 manifest、重写 index，用户只需在 neicanhtml 仓库执行 `git add/commit/push` 即可在 Cloudflare Pages 上看到更新

## 技术实现要点

- **waimaoneican**
  - `src/deploy.py`：`_deploy_to_git_export()` 复制 HTML + JSON，调用 `_build_manifest(export_dir)` 生成 manifest，再调用 `_render_index_html(latest_date, export_dir)` 生成汇总页
  - neicanhtml 目录下需同时保留各期 `neican-YYYY-MM-DD.json`，供 manifest 扫描
- **neicanhtml**
  - `index.html`：单文件，含内联样式与脚本；iframe 嵌入当日 `neican-DATE.html`；日历由 JS 读 `manifest.json` 渲染
  - `manifest.json`：由 deploy 生成，勿手改

## 使用流程

1. 本地运行 `python run_daily.py`（需配置 `NEICAN_GIT_EXPORT_ENABLED=1` 与 `NEICAN_GIT_EXPORT_DIR`）
2. 进入 neicanhtml 仓库执行 `git add . && git commit -m "..." && git push origin main`
3. Cloudflare Pages 自动部署，汇总首页与日历随 manifest 与 index 更新
